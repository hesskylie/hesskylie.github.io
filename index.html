projects

Opus

April 2020

This is an educational virtual piano keyboard I created with three other team members in about two and a half weeks for my bootcamp capstone project. Users can free play on a virtual piano, play a preset song with guided animation, or go on lessons for beginners, complete with exercises to test their skills.

I was responsible for creating the lessons & exercise flow, modifying an existing piano keyboard API to do our bidding, and setting up a progress system for logged in users.

Technology used: React, Redux, Express, Sequelize, React Piano, GreenSock, Node.js, Bootstrap
