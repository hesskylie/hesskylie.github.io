projects

Opus

Opus is an educational virtual piano keyboard I created with three other team members in about two and a half weeks for my Fullstack Academy capstone project. Users can free play on a virtual piano, play a preset song with guided animation, or go on lessons for beginners, complete with exercises to test their skills.

I was responsible for creating the lessons & exercise flow, modifying an existing piano keyboard API to do our bidding, and setting up a progress system for logged in users.

Technology used: React, Redux, Express, Sequelize, React Piano, GreenSock, Node.js, Bootstrap

www.opuslesson.com Github Demo Video

The LAL Store

The LAL Store is a fully functional e-commerce site selling books. I created the app with two other team members in less than two weeks at Fullstack Academy.

I was responsible for all admin features and some user experiences. I found creative solutions to problems such as compiling average ratings for a book & displaying the ratings as stars.

Technology used: React, Redux, Express, Sequelize, Node.js, Bulma, Faker

the-lal-store.herokuapp.com Github

Is It Cancelled?

A comprehensive resource listing events affected by the pandemic. Users can follow events & receive notifications about status changes. I am the sole developer. I created relationships between users and events, sorted the events into easy to follow categories, and assigned color coded status to the events.

Technology used: React, Redux, Express, Sequelize, Node.js, Bulma
